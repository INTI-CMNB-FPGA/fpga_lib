FPGA Lib

Library of HDL snippets (such as cores, procedures and functions) commonly shared between FPGA projects.

This project is mainly developed and mantained by the FPGA division of the Centre of Nano and Microelectronics CMNB of the National Institute of Industrial Technology INTI. Contributions are welcome.

  • We try to develop portable code to be vendor independant.
  • The main used HDL is VHDL (93) and we use GHDL to simulate. You can read vhdl/README.md for more details.
  • Verilog (2001) code and iVerilog Icarus support will be added.
  • We are adding CocoTB for simulations under [testbench](testbench).
  • Documentation is under construction. You can see an HTML version here.
