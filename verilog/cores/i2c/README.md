* Write:
  * MST -> S
  * MST -> ADDR + W
  * SLV -> ACK
  * MST -> DATA
  * SLV -> ACK
  * Repeat 0+ times the last two steps
  * MST -> P
* Read:
  * MST -> S
  * MST -> ADDR + R
  * SLV -> ACK
  * SLV -> DATA
  * MST -> ACK (NACK in the last byte)
  * Repeat 0+ times the last two steps
  * MST -> P
* Combined:
  * P is replaced by another S (Sr)
